Football Badge Memory Game

Game Description

This is a memory game. The aim of the game is to match all the Premier League Club badges, match them all you win.
There are 20 pairs in total.

How To Play

  1. Click a Premier League Logo to reveal a club badge.
  2. Click another Premier League Logo to reveal another badge.
  3. Match the club badges with each other.
  4. The game is over when you have matched all the badges together.
